# Баг-репорт #001
**Заголовок:** [Web] Система принимает значения за пределами шкалы 0-10 (-1, 11), текст ("десять"), символы (#) и пустые значения.

**Шаги воспроизведения:**
1. Перейти по ссылке https://lm.skillbox.cc/qa_tester/module03/homework1/
2. В поле «ОЦЕНКА ПО 10-БАЛЬНОЙ ШКАЛЕ» ввести значения (-1, 11), текст («десять») и символ (#).
3. Нажать кнопку «ОТПРАВИТЬ».

**Ожидаемый результат:** Система сообщила о неправильном заполнении поля «Оценки», выявив ошибку.
**Фактический результат:** Система одобрила неверное заполнение формы и отправила её.

**Серьезность:** S1 (Блокирующий)
**Приоритет:** P1 (Высокий)

**Окружение:** Microsoft Edge v144.0.3719.92, Windows 11 Pro, разрешение экрана 1920x1080
